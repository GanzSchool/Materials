# Ismétlődő feladatok: setInterval

A JavaScript `setInterval` egy beépített függvény, melynek segítségével **ismétlődő** feladatokat hajthatunk végre. Tehát meg tudjuk határozni, hogy egy kódrészlet bizonyos időközönként újra és újra lefusson. 

A `setInterval` efeladata, hogy **rendszeres időközönként** végrehajt egy feladatot anélkül, hogy a fő szálat blokkolná. Ez azt jelenti, hogy képesek vagyunk beállítani azt, hogy bizonyos időközönként újra és újra lefusson a kódrészletünk anélkül, hogy a weboldalunk többi feladatát gátolná.

## Hogyan működik a `setInterval`?

```js
setInterval(callbackFüggvény, időközEzredmásodpercben);
```

- **`callbackFüggvény`**: A függvény, amit ismétlődően meg akarunk hívni.  
- **`időközEzredmásodpercben`**: Megadja, hogy hány ezredmásodpercenként ismétlődjön meg a függvényünk.

Példa: Ha szeretnénk, hogy a függvényünk másodpercenként ismétlődjön meg, akkor 1000 ezredmásodperc beállítását követően a függvényünket a setInterval egy másodpercenként újra és újra meg fogja hívni.

#### HTML kód:

```html
<div id="slideshow">
  <img src="image1.jpg" alt="Kép 1" class="active">
  <img src="image2.jpg" alt="Kép 2">
  <img src="image3.jpg" alt="Kép 3">
</div>
```

#### CSS kód (egyszerű példa):

```css
#slideshow {
  position: relative;
  width: 600px;
  height: 400px;
  overflow: hidden;
}

#slideshow img {
  position: absolute;
  width: 100%;
  height: 100%;
  opacity: 0;
  transition: opacity 1s ease-in-out;
}

#slideshow img.active {
  opacity: 1;
}
```

#### JavaScript kód:

```js
function startSlideshow() {
  const slides = document.querySelectorAll("#slideshow img");
  let currentIndex = 0;
  const totalSlides = slides.length;

  setInterval(function() {
    // Eltávolítjuk az 'active' osztályt a jelenlegi képről
    slides[currentIndex].classList.remove("active");

    // Növeljük az indexet, és ha a végére ért, visszaugrunk 0-ra
    currentIndex = (currentIndex + 1) % totalSlides;

    // Az új képet jelöljük ki aktívnak
    slides[currentIndex].classList.add("active");
  }, 3000); // 3000 ms = 3 másodperc
}

// Indítjuk a diavetítést, amikor betöltődik az oldal
window.onload = startSlideshow;
```
