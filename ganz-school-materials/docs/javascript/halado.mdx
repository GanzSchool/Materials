# JavaScript - Változók

## Elméleti rész

A változók a JavaScript nyelv alapvető építőkövei. Ezek segítségével tárolhatunk adatokat, amelyeket később felhasználhatunk, módosíthatunk vagy kiértékelhetünk a program során.

### Mire használjuk a változókat?

A változók lehetővé teszik:
- Adatok tárolását (számok, szövegek, objektumok, stb.).
- Az adatok újrafelhasználását a program különböző pontjain.
- A program logikájának dinamikus kezelését.

### Hogyan működnek a változók?

A változók deklarálása JavaScriptben történhet az alábbi módokon:

```javascript
let a = 10; // Egy változó létrehozása és értékadás
const b = 20; // Egy állandó változó
var c = 30; // Régi típusú deklaráció (nem javasolt)
```

### A változók típusai

1. **`let`**: Lokális változó, amely újraértékelhető.
2. **`const`**: Állandó értékű változó, nem újraértékelhető.
3. **`var`**: Globális vagy függvény szintű változó (elavult).

#### Hogyan különülnek el egymástól?
- **Hatókör (scope)**: A `let` és `const` blokkszintű, míg a `var` függvényszintű.
- **Újraértékelhetőség**: A `const` értéke nem módosítható, míg a `let` és `var` értéke módosítható.
- **Hosting viselkedés**: A `var` deklarációkat előrehozza a fordító, míg a `let` és `const` nem érhető el a deklaráció előtt.

### Érdekességek
<details>
<summary><b>Miért érdemes kerülni a `var` használatát?</b></summary>
A `var` változók nem blokkszintűek, ami gyakran váratlan viselkedést eredményezhet. Használjunk inkább `let` vagy `const` változókat a biztonságosabb és átláthatóbb kódért.
</details>

<details>
<summary><b>Hogyan válasszunk `let` és `const` között?</b></summary>
Alapvetően mindig használjunk `const`-ot, hacsak nem szükséges az érték újraértékelése, ebben az esetben használjuk a `let`-et.
</details>

### Példatár

#### Feladat 1: Változó deklaráció
Határozz meg egy változót, amely egy számot tárol, és növeld meg az értékét 10-zel!

<details>
<summary><b>Megoldás</b></summary>

```javascript
let szam = 5;
szam += 10;
console.log(szam); // 15
```

</details>

---

#### Feladat 2: Állandó értékek
Deklarálj egy `const` változót egy név tárolására, és próbáld meg módosítani az értékét. Mit tapasztalsz?

<details>
<summary><b>Megoldás</b></summary>

```javascript
const nev = "Anna";
nev = "Péter"; // Hiba: TypeError
console.log(nev);
```
A `const` változók értéke nem módosítható.
</details>

---

#### Feladat 3: Tömbkezelés változókkal
Hozz létre egy tömböt, amely tartalmazza a következő számokat: `[1, 2, 3]`, majd adj hozzá egy új elemet!

<details>
<summary><b>Megoldás</b></summary>

```javascript
let tomb = [1, 2, 3];
tomb.push(4);
console.log(tomb); // [1, 2, 3, 4]
```

</details>

---

#### Feladat 4: Feltételes változókezelés
Hozz létre egy változót, amely tárolja egy szám értékét, és feltételesen növeld meg, ha az érték kisebb, mint 10!

<details>
<summary><b>Megoldás</b></summary>

```javascript
let szam = 8;
if (szam < 10) {
  szam++;
}
console.log(szam); // 9
```

</details>

#### Feladat 5: Sztringek és változók
Tárolj egy szöveget egy változóban, majd adj hozzá egy új szövegrészt!

<details>
<summary><b>Megoldás</b></summary>

```javascript
let szoveg = "Üdvözlöm ";
szoveg += "az olvasót!";
console.log(szoveg); // Üdvözlöm az olvasót!
```

</details>
